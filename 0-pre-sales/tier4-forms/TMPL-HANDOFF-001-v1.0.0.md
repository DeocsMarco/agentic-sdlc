## Purpose

This template structures the project handoff meeting and documentation from Account Manager to Project Manager. It ensures all critical project knowledge, context, relationships, and materials are transferred systematically to enable successful project execution.

**Completed By**: Account Manager (AM) and Project Manager (PM) collaboratively
**Time Required**: 60-90 minutes (handoff meeting)

---

## Instructions

1. **Schedule handoff meeting** with AM, PM, and PM Lead (60-90 minutes)
2. **AM prepares** all source documents and answers to sections 1-7 before meeting
3. **During meeting**, AM and PM complete this template collaboratively
4. **PM asks clarifying questions** throughout; goal is PM fully understands project
5. **PM signs acceptance** at end confirming readiness to take over project
6. **File completed handoff document** in project folder for reference

**Important**: This is NOT a one-way information dump. PM should actively engage, ask questions, and challenge assumptions. Handoff is successful when PM can independently answer the 4 key questions in Section 8.
