## Purpose

This form guides the allocation of resources (team members) to project phases with FTE calculation using the formula: FTE = Effort ÷ Duration ÷ Utilization. It ensures realistic resource planning by accounting for part-time assignments, availability constraints, and skill requirements.

**Used By**: Project Manager (with Resource Manager input)
**Time Required**: 45-60 minutes

---

## Methodology Overview

Resource allocation converts effort estimates (from WORK-PM-004) into team size and duration by mapping resources to phases and calculating Full-Time Equivalent (FTE) requirements. The key formula ensures realistic staffing based on available time and utilization rates.

**Key Principles**:
1. **Utilization <100%**: People are never 100% billable (meetings, admin, learning, vacation = ~80% typical)
2. **Phase-based allocation**: Different roles needed in different phases (designers in Design, developers in Dev, etc.)
3. **Named resources when possible**: Assign specific people, not generic "developer" roles
4. **Skill matching**: Ensure resources have required skills for assigned tasks

**Steps**:
1. Review effort estimate from WORK-PM-004 by phase
2. Determine phase duration based on schedule constraints
3. For each phase, identify required roles and skill levels
4. Calculate FTE per role using formula: FTE = Effort ÷ Duration ÷ Utilization
5. Assign named resources (or identify gaps)
6. Validate total resource availability and cost

## Reference Tables

### Table 1: Standard Utilization Rates

| Resource Type | Typical Utilization | Explanation |
|---------------|--------------------|--------------|
| **Full-time dedicated** | 80% | 32h/week billable (8h for meetings, admin, email, etc.) |
| **Full-time shared (2-3 projects)** | 60% | 24h/week billable per project (rest on other projects) |
| **Part-time (50%)** | 40% | 16h/week billable (50% allocation × 80% utilization) |
| **Contractors (remote)** | 90% | 36h/week billable (less meeting overhead) |
| **Executives/Advisors** | 10-20% | 4-8h/week (advisory role only) |

**Usage**: Select appropriate utilization rate based on resource type. Never use 100% unless resource is literally doing nothing but this project (rare).

---

### Table 3: FTE Calculation Examples

| Scenario | Effort | Duration | Utilization | FTE Calculation | Result |
|----------|--------|----------|-------------|----------------|--------|
| **Full-time dev** | 320h | 8 weeks | 80% | 320 ÷ (8×40) ÷ 0.8 = 320 ÷ 320 ÷ 0.8 | 1.25 FTE |
| **Part-time QA** | 160h | 8 weeks | 40% | 160 ÷ 320 ÷ 0.4 | 1.25 FTE |
| **Contractor** | 200h | 10 weeks | 90% | 200 ÷ 400 ÷ 0.9 | 0.56 FTE |

**Formula Breakdown**:
- **Effort**: Total person-hours for this role in this phase (from WORK-PM-004)
- **Duration**: Phase length in weeks
- **Hours Available**: Duration × 40 hours/week
- **Utilization**: % of available hours actually billable to project
- **FTE = Effort ÷ (Duration × 40) ÷ Utilization**

## Common Resource Allocation Errors & Solutions

| Error | Cause | Solution |
|-------|-------|----------|
| FTE >1.5 but expecting 1 person | Utilization set too low or duration too short | Recalculate: If FTE=1.5, need 1.5 people or extend duration by 50% |
| Everyone at 100% utilization | Unrealistic planning; no time for meetings/admin | Use 80% for full-time, 60% for shared resources |
| Same person allocated >100% total FTE | Double-booking across projects | Check resource calendar; negotiate priorities or add resources |
| Generic roles instead of named people | Resource Manager not engaged | Confirm availability with Resource Manager before finalizing plan |
| Skill mismatch (junior assigned complex work) | Availability-driven vs. skill-driven staffing | Either upskill junior (add training time) or find appropriate skill level |
